# بوابة مشاريع التخرج - كلية الحاسبات والمعلومات (MUST)

هذا هو المشروع المكتمل لبوابة مشاريع التخرج الخاصة بكلية الحاسبات والمعلومات بجامعة مصر للعلوم والتكنولوجيا. تم بناء المشروع باستخدام Angular وبتصميم كامل يدعم اللغة العربية ونظام الـ RTL.

## المميزات الرئيسية
- **نظام كامل للبحث وتصفح أفكار المشاريع**: يمكن للطلاب تصفح الأفكار وتصنيفها والبحث عنها.
- **استمارة التخرج القابلة للطباعة**: استمارة مصممة لنظام A4 تحتوي على كافة البيانات المطلوبة والاعتمادات.
- **لوحة تحكم (Dashboard)**: تعرض إحصائيات عامة عن مشاريع التخرج والأخبار الهامة.
- **نظام المصادقة (Auth)**: يحتوي على صفحات تسجيل الدخول وإنشاء الحساب مع دعم الأدوار (طالب، دكتور، مدير).
- **مكتبة القوالب**: لتحميل النماذج الإرشادية والقوالب الرسمية.
- **دعم كامل للغة العربية (RTL)**: استخدام خط Tajawal وتصميم متوافق مع الاتجاه من اليمين لليسار.

## كيفية التشغيل
1. تأكد من تثبيت Node.js و Angular CLI.
2. انتقل لمجلد المشروع:
   ```bash
   cd must-fci-portal
   ```
3. قم بتثبيت التبعيات:
   ```bash
   npm install
   ```
4. قم بتشغيل خادم التطوير:
   ```bash
   ng serve
   ```
5. افتح المتصفح على `http://localhost:4200`.

## ملاحظات فنية
- تم استخدام **Angular 17** مع معمارية **NgModule** (standalone = false).
- تم استخدام **SCSS** للتنسيقات.
